# Expression languages
## C
### Operators supported
* [ ]
* ++ -- (postfix)
* & \* + - ~ !
* ++ -- (prefix)
* \* / %	
* \+ \-
* < > <= >=
* == !=	
* &	
* ^	
* |	
* &&	
* ||
* ? :	(Conditional-expression)

### Variable types supported
* int
* double
* bool
* DateTime
